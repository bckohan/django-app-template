## Supported Versions

Only the latest version [![PyPI version](https://badge.fury.io/py/{{cookiecutter.project_slug}}.svg)](https://pypi.python.org/pypi/{{cookiecutter.project_slug}}) is supported.

## Reporting a Vulnerability

If you think you have found a vulnerability, and even if you are not sure, please [report it to us in private](https://github.com/{{cookiecutter.github_owner}}/{{cookiecutter.project_slug}}/security/advisories/new). We will review it and get back to you. Please refrain from public discussions of the issue.
